Data Types

UFO

Columns: datetime, city, state, country, shape, duration(seconds)

Rows: items

- datetime: datatype: attributes(quantitative)

- city: datatype: attributes(categorical)

- state: datatype: attributes(categorical)

- country: datatype: attributes(categorical)

- shape: datatype: attributes(categorical)

- duration(seconds): datatype: attributes(quantitative)

Airports

Columns: Airport_name, Latitude, Longitude

Rows: items

- Airport_name: datatype: attribute(categorical)

- Latitude: datatype: position

- Longitude: datatype: position

Google-Web

Columns: FromNodeID, ToNodeId

Rows: items

- FromNodeId: datatype: link

- ToNodeId: datatype: link
